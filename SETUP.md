# Setup Guide

This repository contains Machine Learning projects and experiments using various ML frameworks.

## Dependencies

All required dependencies are listed in `requirements.txt`. The main libraries included are:

### Core Data Science Libraries
- **NumPy**: Numerical computing with arrays and matrices
- **Pandas**: Data manipulation and analysis
- **SciPy**: Scientific computing and optimization

### Machine Learning
- **scikit-learn**: Classical machine learning algorithms

### Deep Learning Frameworks
- **PyTorch**: Deep learning framework with dynamic computation graphs
- **TorchVision**: Computer vision utilities for PyTorch
- **TorchAudio**: Audio processing utilities for PyTorch
- **TensorFlow**: Deep learning framework by Google

### Visualization
- **Matplotlib**: Plotting and visualization
- **Seaborn**: Statistical data visualization
- **Plotly**: Interactive plotting

### Jupyter Ecosystem
- **Jupyter**: Interactive notebooks
- **JupyterLab**: Next-generation notebook interface
- **IPython**: Enhanced interactive Python shell

### Utilities
- **tqdm**: Progress bars
- **Pillow**: Image processing
- **OpenCV**: Computer vision library

## Installation

To install all dependencies, run:

```bash
pip install -r requirements.txt
```

## Verification

To verify that all dependencies are installed correctly, run:

```python
import numpy as np
import pandas as pd
import torch
import tensorflow as tf
import sklearn
import matplotlib
import seaborn
import cv2

print("All dependencies installed successfully!")
```

## Getting Started

You can now start building Machine Learning projects using:
- Jupyter notebooks for interactive development
- PyTorch or TensorFlow for deep learning
- scikit-learn for classical ML algorithms
- Pandas for data manipulation
- Matplotlib/Seaborn for visualization
